HTML (HyperText Markup Language)

O HTML é a espinha dorsal de uma página web. Ele fornece a estrutura básica, permitindo que você defina títulos, parágrafos, listas, links e outros elementos. Utilizando tags HTML, você marca o conteúdo para que o navegador entenda como apresentá-lo. HTML é estático por natureza, o que significa que define a estrutura e o conteúdo, mas não lida com interatividade ou estilo.

Nível Inicial:


Introdução HTML:

  • Compreenda a estrutura básica de uma página HTML.
  • Aprenda sobre tags, elementos e atributos HTML.

    • Elementos Básicos:

  • Familiarize-se com elementos essenciais como <head>, <body>, <p>, <h1>- <h6>, <ul>, <ol>, <li>, <a>, etc.
  • Entenda a semântica HTML.

    • Formulários:

  • Aprenda a criar formulários usando elementos como <form>, <input>, <textarea>, <select>, e outros.
  • Experimente diferentes tipos de entrada e validações.

Nível Intermediário:


Multimídia:

  • Inclua imagens com <img> e vídeos com <video>
  • Explore elementos de áudio com <audio>.

    • Tabelas e Listas Avançadas:

  • Crie tabelas usando <table>,<tr>,<td>, etc.
  • Utilize listas aninhadas e personalizadas.

    • HTML Semântico:

  • Compreenda a importância dos elementos semânticos, como <header>, <nav>, <main>, <article>, <section>, <footer>, etc.
  • Utilize esses elementos para estruturar melhor o conteúdo.

Nível Avançado:


Acessibilidade:

  • Aprenda sobre boas práticas de acessibilidade.
  • Utilize atributos como aria-* para melhorar a acessibilidade.

    • Formulários Avançados:

  • Explore campos de formulário avançados, como <fieldset>, <legend>, <input type='data'>, <input type='range'>, etc.

    • Integração com CSS:

  • Entenda como estilos CSS podem ser aplicados aos elementos HTML.
  • Explore classes, IDs e seletores.

    • Meta Tags e SEO:

  • Utilize meta tags para melhorar o SEO.
  • Entenda a importância do atributo alt em imagens.

    • Prática e Projetos:

  • Construa páginas web simples para praticar os conceitos.
  • Crie formulários funcionais. Adote a semântica HTML em projetos reais.

    As melhores trilhas para se guiar atraves da longa estrada da ciência da computação

    CSRoadMap é uma ferramenta de aprendizagem para auxiliar e mostrar um caminho para guiar futuros(as) cientistas da computação que não sabem por onde começar a estudar. Como em um jogo RPG, queremos mostrar aos estudantes que existem inumeras trilhas e rotas a seguir ao entrar na area da computação, porém essa escolha deve vir diretamente do player, ou seja, você!

    Desenvolva seu conhecimento e codifique seu futuro

    Conforme sua caminhada de estudos, cada tecnologia terá um peso maior. Iremos te indicar foruns, videos, bibliotecas, repositorios, desafios e referencias para aprimorar seus conhecimentos. Porém a melhor parte e que você mesmo pode nos ajudar a melhorar nossa plataforma, atraves de nossa politica Open Source!

    De Devs para Devs

    CSRoadMap aspira a ser o parceiro de confiança de cada estudante de Ciência da Computação, guiando-os em sua jornada acadêmica e preparando-os para os desafios do mundo profissional. Ao oferecer recursos de alta qualidade, práticos e interativos, a plataforma visa capacitar os estudantes a alcançarem todo o seu potencial na área de Ciência da Computação.
